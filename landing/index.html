Manuscript annotation and handwritten text recognition

Segment folio images, run line-level HTR, and export training-ready data. Built for Greek, Syriac, and related historical scripts in the NOMOS research ecosystem.

Features

Built for manuscript HTR workflows

  • Page editor

    Draw line regions on each folio, pair transcriptions with segments, and compare ground truth against model output on the same canvas.

  • Segmentation and transcribe

    Run layout segmentation and line OCR from the editor. Switch models per task and keep results as separate transcription layers.

  • Local or cloud inference

    Use hosted inference by default, or install the Inference Helper to run OCR and segmentation on your CPU with models cached on disk.

  • Projects and export

    Organize work into projects and documents, publish live pages, and export line crops plus transcriptions for training pipelines.

About

Open research tooling

Nomicous is the annotation platform behind the NOMOS project: a hosted editor, open HTR models on Hugging Face, and an optional local inference helper. Weights sync from the registry so new models do not require reinstalling the helper.
